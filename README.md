Projeto da Faculdade UTFPR - Disciplina: Sistemas Distribuidos

Título do Projeto: Resolução Sequencial e Paralela do Problema das N Rainhas

Descrição:

Este projeto tem como objetivo desenvolver uma solução para o Problema de N Rainhas utilizando técnicas de computação sequencial e paralela. O foco do projeto é minimizar o tempo de execução do algoritmo, distribuindo o processamento entre múltiplas threads (no caso da solução paralela).
O Problema de N Rainhas consiste em posicionar N rainhas em um tabuleiro de xadrez de NxN de forma que nenhuma rainha ataque outra. Este projeto propõe uma generalização do problema clássico das 8 rainhas.

Implementação:

 - Solução Sequencial: Uma implementação inicial para validar a lógica do algoritmo e servir como linha de base para comparações de desempenho.

 - Solução Paralela: Após validar a solução sequencial, foi desenvolvido e otimizada uma solução paralela para o problema.
